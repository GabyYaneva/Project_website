Personal Portfolio Website
This repository contains the source code for my personal portfolio website. It serves as a central hub to showcase my software engineering projects, technical skills, and professional journey.

🚀 Overview
The goal of this project was to create a clean, responsive, and professional space to document my work in System Programming, Machine Learning, and Web Development.

🛠️ Built With
HTML5: For structured and semantic web content.

CSS3: Custom styling with a focus on responsiveness and modern UI/UX principles.

JavaScript: (Optional, ако си добавила) For interactive elements and smooth navigation.

✨ Key Features
Project Showcase: A dedicated section highlighting my work, including my SVM implementation, C Web Crawler, and HTTP Client.

Responsive Design: Fully optimized for different screen sizes, from desktops to mobile devices.

Clean Architecture: Well-organized code structure for easy maintenance and updates.

Contact Information: Easy ways for recruiters and collaborators to get in touch.

📂 Structure
index.html – The main landing page.

css/ – Stylesheets defining the visual identity.

assets/ – Images, icons, and professional photos.

projects/ – (Ако имаш отделни страници) Detailed descriptions of individual projects.

📈 Performance
The site is built with a "minimalist" mindset to ensure:

Fast Loading Times: No heavy frameworks or unnecessary bloat.

SEO Friendly: Proper use of tags for better search engine visibility.
